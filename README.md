# Arch Rice

Run on a fresh Arch install to get my cool rice

## How to use

```bash
sudo pacman -S git
git clone https://gitlab.com/squareboi104/arch-rice.git
sh ./arch-rice/arch-rice.sh
```
OR

```bash
sudo pacman -S wget
wget -LO https://gitlab.com/squareboi104/arch-rice/-/raw/main/arch-rice.sh
sh ./arch-rice.sh
```
